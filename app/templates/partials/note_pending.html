{# Background research-note polling partial (STABLE outer poller pattern). The OUTER #note-poll element carries the poll trigger and is NEVER replaced while running — it swaps only its INNER content (#note-poll-inner). This is far more reliable in HTMX 1.9.x than a self-replacing (outerHTML) poller, which can silently stop polling after the first swap. While the job runs, GET /analysis/note/status returns just the spinner inner HTML (note_pending_inner.html) which swaps into #note-poll-inner. When the job finishes, the status route returns the final note wrapped with an hx-swap-oob that REPLACES the whole #note-poll (removing the trigger) so polling stops and the report is shown. #}
